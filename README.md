# tugas-uas-quiz

Project ini adalah web kuis sederhana berbasis HTML, CSS, dan JavaScript
yang dibuat untuk latihan

# Fitur
 Pertanyaan pilihan ganda
 Penilaian otomatis
 Tampilan interaktif

# Teknologi
1. HTML
2. CSS
3. JavaScript

# Alur logika 
1. Soal tampil ke halaman
2. User memilih jawaban
3. JavaScript mengecek jawaban
4. Skor akan dihitung
5. nilai akan ditampilkan
6. user lulus/tidak

# Cara Menjalankan
Buka file index.html menggunakan browser

# Catatan
Project ini dibuat untuk tugas ujian sekaligus untuk pembelajaran
